Push vs Pull

Один pipeline (filter чётные → ×2). Push: RxJS emit'ит в subscribe. Pull: Iterator читает коллекцию; Promise — те же шаги через .then, одно значение.

Push RxJS

from(source).pipe(
  filter(x => x % 2 === 0),
  map(x => x * 2),
  take(3)
).subscribe(onNext)
…

Pull

Iterator helpers N values

Iterator.from(source)
  .filter(x => x % 2 === 0)
  .map(x => x * 2)
  .take(3)
  .toArray()
…
